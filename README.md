## 技术

框架：vue、webpack

插件：vuex、babel、sass、bootstrap、mint-ui、mui等

## 运行项目

```shell
git clone git@github.com:ForeManWang/vue-shopping.git --depth 1

yarn install
```

在`npm run dev`之前先修改你自己需要运行在的端口号，在**package.json**中修改**dev**中的配置就行

```shell
npm run dev
```
